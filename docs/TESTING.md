Test the site responsively at these viewport sizes:

| Size | Device | Layout |
|------|--------|--------|
| 320px | Old phone | 1 column |
| 480px | Modern phone | 1-2 columns |
| 768px | Tablet | 2-3 columns |
| 1024px | Laptop | 3-4 columns |
| 1280px | Desktop | 4 columns |
| 1536px | Wide | Full |

## Chrome DevTools
1. Press `F12`
2. Press `Ctrl+Shift+M` 
3. Enter viewport width
4. Test at sizes above

## What to Check
- Text readable at all sizes
- No horizontal scrolling
- Grid layouts responsive
- Padding/spacing appropriate
- Images scale correctly
- Touch targets 44x44px minimum
